# Games Library

Единая библиотека игровых определений, чтобы файлы конкретных игр были отделены от ядра платформы.

## Структура
- `src/definitions.mjs` — каталог игр и фабрики начального state.
- `assets/` — статические игровые данные/темы.

Правило: все новые game-specific определения добавляются в `libraries/games/`.
