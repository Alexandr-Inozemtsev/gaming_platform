# Big Walker (Unity module)

Минимальный production-ready каркас настольной игры «Большая Бродилка» для Unity.

## Что реализовано
- Бросок кубика с анимацией и задержкой (корутина).
- Пошаговая система на 2–6 игроков.
- Движение фишки по клеткам с плавной анимацией «шаг за шагом».
- Эффекты клеток:
  - `Forward` — шаг вперёд на N клеток;
  - `Backward` — шаг назад на N клеток;
  - `SkipTurn` — пропуск следующего хода;
  - `ExtraTurn` — дополнительный ход.
- Победа при достижении финальной клетки (можно настроить правило «только точное попадание»).
- API для подключения кастомного UI под ваш визуальный дизайн (как на референсе).

## Структура
- `Assets/Scripts/Config` — ScriptableObject-конфиги.
- `Assets/Scripts/Core` — игровой цикл, игроки, логика ходов.
- `Assets/Scripts/Board` — клетки и путь.
- `Assets/Scripts/Animation` — анимации движения и кубика.
- `Assets/Scripts/UI` — интерфейс для UI-связывания.

## Быстрое подключение в ваш проект
1. Установите Unity 2022.3 LTS+.
2. Создайте Unity-проект (URP рекомендуется).
3. Скопируйте папку `Assets/Scripts` из этого модуля в ваш Unity-проект.
4. На сцене создайте:
   - `GameController` (с компонентом `BigWalkerGameController`),
   - `BoardRoot` (с компонентом `BoardPath`),
   - по фишке (`TokenMover`) на игрока,
   - UI-слой с реализацией `IBigWalkerGameView`.
5. Создайте `BigWalkerGameConfig` через **Create → BigWalker → Game Config** и заполните поля.
6. Свяжите ссылки в инспекторе и запустите Play Mode.

## Интеграция в существующий монорепозиторий
В этом репозитории добавлен модуль `apps/unity-big-walker` как отдельная директория с кодом и инструкциями.
Это не ломает существующие Node/Flutter пайплайны, пока Unity-проект собирается отдельно.
