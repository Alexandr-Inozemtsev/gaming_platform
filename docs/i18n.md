# Локализация RU/EN (Prompt O)

## Назначение
Документ описывает реализацию i18n в MVP: словари по ключам, plural-правила и базовое форматирование чисел/валют.

## Параметры
- `LANGS=["ru","en"]`
- `DEFAULT=ru`

## Реализация
- Словари находятся в `apps/mobile/lib/i18n/strings.dart`.
- В `AppState` используются обёртки:
  - `t(key)` — обычная строка;
  - `tp(key, count)` — plural-строка;
  - `formatNumber(value)` — формат числа по языку;
  - `formatCurrency(value)` — формат валюты по языку.

## Plural-правила
- `ru`: `one/few/many` с учётом исключений 11-14.
- `en`: `one/other`.

Ключи plural-строк задаются в формате:
- `<key>.one`
- `<key>.few`
- `<key>.many`
- `<key>.other`

## Acceptance checklist
- Основные экраны (`Home/Catalog/Room/Create/Store/Profile/Settings`) не используют жёсткие UI-строки — тексты берутся через ключи i18n.
- Переключение языка в Settings мгновенно обновляет UI, так как все виджеты читают тексты через `AppState.t(...)`.
- Добавлена отдельная документация `docs/i18n.md`.

## Минимальное форматирование
- `formatNumber`:
  - `ru`: пробелы как разделители тысяч, запятая в дробной части;
  - `en`: запятые как разделители тысяч, точка в дробной части.
- `formatCurrency`:
  - `ru`: `<число> ₽`
  - `en`: `$<number>`
