# Назначение: спецификация централизованных дизайн-токенов.

## Цвета
- Base: `bg/base #0F1115`, `bg/elevated-1 #171A21`, `bg/elevated-2 #1E2430`, `bg/panel #232A38`.
- Text: `text/primary #F5F7FA`, `text/secondary #B9C1CC`, `text/muted #7E8794`, `text/inverse #0F1115`.
- Accent primary: `#2EC5B6 / #37D6C6 / #21A89B / #183C3A`.
- Accent secondary: `#F2B24D / #FFC76A / #D89A36 / #4A3720`.
- Semantic: success/warning/error/info.
- Strokes: soft/default/strong.
- Gameplay: active/opponent/danger/reward.

## Типографика
- Heading family: Manrope (fallback-friendly).
- Body family: Inter.
- Tokens: `display/lg`, `h1`, `h2`, `h3`, `body/md`, `body/sm`, `label`, `caption`.

## Пространство и форма
- Spacing: 4/8/12/16/20/24/32/40.
- Radius: 8/12/16/20/24/999.
- Elevation: level1/2/3 + glow primary/secondary.
- Opacity: disabled/subtle/overlay.
- Z-index: base/hud/sheet/overlay/modal/toast.

## Motion tokens
- Duration: 120/180/240/320ms.
- Easing: standard/emphasized/exit.

## Layout constants
- Size classes: compact (<900w), regular (900..1179), large (>=1180).
- Panel widths: 280/320/360.
- Safe-area padding: always media query aware.
