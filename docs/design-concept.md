# Назначение: единая визуальная концепция tactical tabletop lounge для MVP.

## Design concept summary
- **Тон**: премиальный цифровой игровой стол с тёмной базой, мягкими teal/amber акцентами и чистым технологичным UI.
- **Принципы**: landscape mobile first, минимальный шум, читаемость при долгих сессиях, тактильность карточек/жетонов/панелей.
- **UX-фокус**: gameplay area всегда приоритетнее вторичных блоков; вторичные панели сворачиваются/уходят в overlay в compact landscape.
- **Системность**: все стили задаются только через token-layer и reusable UI-kit.
- **Платформенность**: единый язык для iOS/Android с учётом safe areas, camera cutouts, жестов и touch targets 44dp+.
